LipidQuest: Interactive Lipid Biology Game
LipidQuest is an educational game designed to teach key concepts in lipid biology through interactive, game-based learning. The project combines biochemical principles with engaging gameplay mechanics to help users understand lipid classification, nutrition, membrane dynamics, and molecular interactions.
Overview: The game is structured as a series of mini-games, each focused on a different learning objective within lipid biology. Players progress through levels that gradually increase in complexity, reinforcing both conceptual understanding and application.

Mini-Games: 
1. Lipid Classification
Learning Goal: Identify and classify the eight major lipid classes

Gameplay: A match-3 style game inspired by Candy Crush, where players match lipid structures to clear tiles from a grid.

Lipid Classes Included:
Fatty acyls
Glycerolipids
Glycerophospholipids
Sphingolipids
Sterols
Prenols
Saccharolipids
Polyketides

Mechanics:
Match 3 or more of the same lipid class to clear tiles
Chain reactions and combos increase score
Power-Ups:
Headgroup Bomb: clears all phospholipids
Sterol Ring: clears entire rows
Tail Length Bonus: longer fatty acids clear more tiles
Saturation-based effects (in progress)

2. Lipids in Food & Nutrition
Learning Goal: Understand dietary sources and nutritional roles of lipids

Gameplay: A sorting and optimization puzzle where players build balanced meals.

Mechanics:
Foods (e.g., olive oil, salmon, butter, nuts) move along a conveyor belt
Each food breaks into lipid components:
Saturated fats
Unsaturated fats
Trans fats
Cholesterol
Omega-3 / Omega-6

Player Tasks:
Sort foods into categories:
Energy storage
Essential nutrients
Limit intake
Balance macronutrient goals

Dynamic Effects:
Excess saturated fats slow gameplay (metabolic penalty)
Omega-3s provide regeneration bonuses in later levels

3. Lipids in Membranes (in progress)
Learning Goal: Explore how lipid composition affects membrane structure and function

Gameplay: A physics-inspired membrane builder where players construct lipid bilayers.

Mechanics:
Build membranes using:
Phosphatidylcholine
Phosphatidylethanolamine
Sphingomyelin
Cholesterol

Adjust:
Headgroup size
Tail saturation
Cholesterol content
Temperature and mechanical stress

Key Concepts:
Membrane fluidity vs rigidity
Permeability and stability
Curvature (vesicles vs flat membranes)

4. Lipid-Protein Interactions (in progress)
Learning Goal: Understand how proteins recognize and bind lipid molecules

Gameplay: A docking and recognition puzzle involving molecular interactions.

Mechanics:
Proteins contain:
Charged residues
Hydrophobic pockets
Aromatic regions
Lipid ligands (e.g., cholesterol, fatty acids) must be correctly placed
Interaction Rules:
Hydrogen bonding (headgroups)
Hydrophobic matching (tails)
Shape complementarity

Outcomes:
Correct binding → functional signaling
Incorrect binding → mislocalization or failure

Status: 
This project is actively under development.
Mini-games 3 (Membrane Builder) and 4 (Lipid-Protein Interactions) are currently in progress, with core mechanics and designs being implemented.

Tech Stack:
Python
Pygame
RDKit (for molecular structure visualization)

Goal:
The goal of LipidQuest is to make complex biochemical concepts more intuitive and accessible through interactive simulation and gameplay, particularly for students learning lipid biology for the first time.
